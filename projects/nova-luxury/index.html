Overview

What it is

Nova Luxury Furniture is a complete furniture e-commerce platform designed and developed from concept to production. It combines a polished storefront with a robust Node.js and MongoDB backend, focusing on performance, scalability, and a smooth shopping experience.

Highlights

Key features

  • Full furniture e-commerce platform built from concept to production.
  • 30+ REST API endpoints for product, category, and order management.
  • Secure REST APIs, authentication, authorization, and database integration (Node.js + MongoDB Atlas).
  • Deployed and maintained on Cloudflare and Railway with a focus on performance and scalability.
